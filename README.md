# FreeIran Sources

This repository contains publicly accessible **evidence and informational resources about the situation in Iran**.

The goal of this website is to provide **clear, factual, and well-organized documentation** that allows readers — including policymakers, journalists, and members of the public — to better understand the situation of the Iranian people.

The materials included here may contain:

- videos documenting protests
- photographs and visual evidence
- official reports and documents
- testimonies
- statistics and timelines

All information should aim to be **factual, verifiable, and respectful of the Iranian people**.
